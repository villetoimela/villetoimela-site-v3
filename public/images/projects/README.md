# Project Images

Add your project images to this folder.

## Image naming convention:
- Use descriptive names like: `project-name.jpg` or `project-name.png`
- Recommended format: JPG or PNG
- Recommended size: 1200x675px (16:9 ratio) or similar

## Usage:
Images in this folder can be referenced in `/data/projects.ts` as:
```typescript
image: '/images/projects/your-image-name.jpg'
```

## ProjectsPreview component:
Update the `projects` array in `components/ProjectsPreview.tsx` to use your images.
